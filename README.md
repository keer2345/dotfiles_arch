## 如何同步 dotfiles

- [yadm](https://github.com/yadm-dev/yadm)
- [Yadm Getting Started](https://yadm.io/docs/getting_started#)

```sh
yadm init
yadm add .zshrc
yadm commit -a

yadm remote add origin <url>
yadm push -u origin <local branch>:<remote branch>

yadm pull
yadm status
```
